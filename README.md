# CVS Health (cvs-health)

CVS Health is a Fortune 50 healthcare services and retail pharmacy company. Core operating units include CVS Pharmacy (retail and mail-order pharmacy), CVS Caremark (pharmacy benefit management / PBM), Aetna (health insurance, Medicare, Medicaid, dental, vision), MinuteClinic (walk-in clinics), Oak Street Health (primary care for Medicare patients), and Signify Health (in-home health evaluations). CVS Health does not currently operate a public developer portal or generally available REST API; partner integrations flow through contracted channels (NCPDP SCRIPT, X12 EDI, HL7 / FHIR endpoints required by the CMS Interoperability Rule).

## APIs

### CVS Health Partner Integrations

CVS Health does not publish a unified public REST API or developer portal. Pharmacy, PBM, and Aetna integrations are conducted via contracted partner channels: NCPDP SCRIPT for e-prescribing, NCPDP Telecom for claims, X12 EDI for healthcare transactions, and HL7 / FHIR endpoints required by CMS Interoperability and Patient Access regulations.

- **Human URL:** https://www.cvshealth.com/social-responsibility/digital-health

## Capabilities

- Retail and mail-order pharmacy operations
